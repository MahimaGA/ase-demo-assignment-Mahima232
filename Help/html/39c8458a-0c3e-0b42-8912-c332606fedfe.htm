      ASEproject Namespace

      The ASEproject namespace contains classes and functionality

      Classes

      AppArray Represents a custom implementation of the Array class that also implements ICommand. This class processes array parameters for commands in the application.
      AppCanvas Represents the canvas where drawings are made.
      AppCommandFactory CommandFactory class for adding more commands
      AppElse Represents the 'Else' command in the application. Inherits from the Else class and modifies its behavior.
      AppEnd Represents the 'End' command in the application. Inherits from the End class and modifies its behavior.
      AppFor Represents the 'For' command in the application. Inherits from the For class and modifies its behavior.
      AppIf Represents the 'If' command in the application. Inherits from the If class and modifies its behavior.
      AppInt Represents the 'Int' command in the application. Inherits from the Int class and modifies its behavior. Implements the ICommand interface.
      AppMethod Represents a custom application-specific method that extends the functionality of the Method class. Provides additional methods for resetting or decreasing internal counters.
      AppParser Represents a parser for the application. Inherits from the Parser class and implements the IParser interface.
      AppPeek Represents the "Peek" command in the application. Inherits from the AppArray class and implements the command functionality.
      AppPoke Represents the "Poke" command in the application. Inherits from the AppArray class and implements the command functionality.
      AppReal Represents the "Real" command in the application. Inherits from the Real class and implements the command functionality.
      AppStoredProgram Represents a stored program with conditional command stack functionality. Inherits from the StoredProgram class and provides custom implementations for managing commands.
      AppWhile Represents a custom implementation of the While class for the ASE project. Inherits from the base While class and provides additional functionality.
      Clear Represents the "Clear" command that clears the canvas and sets the background to grey.
      Form1 represents the main form
      Reset represents a command that resets pen position and color
      Tri represents a command to draw a triangle on canvas
      Write writes text in canvas
